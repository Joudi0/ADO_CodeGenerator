# Joudi's 3-Tier Code Generator 🚀

أداة تفاعلية مبنية بلغة C# لتوليد أكواد طبقات الوصول للبيانات (DAL) وطبقة الأعمال (BLL) تلقائياً لمشاريع ADO.NET التي تعتمد على معمارية 3-Tier. الأداة مصممة لتوفير ساعات من كتابة الكود اليدوي المتكرر.

## الميزات ✨
- توليد دوال الـ CRUD كاملة (Add, Update, Delete, GetAll, Get By ID).
- واجهة مستخدم تفاعلية بالكونسول (CLI) بتصميم عصري باستخدام مكتبة Spectre.Console.
- التحقق التلقائي من وجود الجداول والأعمدة في قاعدة البيانات قبل توليد الكود لحماية المشروع من الأخطاء.
- كتابة كلاسات جاهزة للاستخدام الفوري مع التعامل الآمن مع الـ DBNull والـ Parameters لمنع ثغرات SQL Injection.

## كيف تستخدم الأداة؟ 🛠️
1. افتح كلاس `clsHelper` وقم بتعديل متغير `connectionString` ليتوافق مع إعدادات قاعدة البيانات الخاصة بك.
2. شغل الأداة (Run).
3. اكتب اسم الجدول الذي تريد توليد الكود له.
4. اتبع التعليمات التفاعلية على الشاشة لاختيار الدوال التي تريد إنشاءها.
5. انسخ الكود المولد وألصقه في مشروعك!
